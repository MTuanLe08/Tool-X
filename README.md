2. **Open Termux:**
   - Locate the Termux app and open it.

3. **Update Packages:**
   - In the Termux terminal, type the following command and press Enter:
     ```
     pkg update && pkg upgrade
     ```

4. **Install Required Packages:**
   - Type this command to install necessary packages:
     ```
     pkg install git
     ```

5. **Download Tool-X:**
